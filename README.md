# Sschlesier Mdserver

## How do I install these formulae?

`brew install sschlesier/mdserver/<formula>`

Or `brew tap sschlesier/mdserver` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sschlesier/mdserver"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
